Configuration

Configuration options

The configuration options are available after pressing the MENU button from the main screen. The following preferences can be configured:

Do not switch off the display

When ticked, the screensave won't switch off the display.

Move the map so that current position is always in the center

When ticked, the map will be centered on the current location after acquiring a new position update.

Saving and reading point collections

The current collection of points can be saved by pressing the SAVE button. It is also automatically saved after introducing any changes (adding, editing, removing points). The name of the current collection is displayed in the “Name of the current collection” field. A new collection can be created by editing the name and pressing “SAVE”. Switching between collections is done by choosing from the expandable list “Collections”.

Pressing “REMOVE” deletes the current collection.

Language choice

The language can be switched between English and Polish.
